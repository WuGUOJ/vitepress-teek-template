# vitepress-teek-template

## 简介
基于 `vitepress-theme-teek` 主题的基础模板，你可以基于此模板快速创建自己的文档。

参考链接：
- [vitepress](https://vitepress.dev/zh/)
- [vitepress-theme-teek](https://vp.teek.top/)

## 内置功能
- PlantUML
- Mermaid
- echarts

## 开始
### 安装依赖
```
pnpm install
```
### 运行
```
pnpm run dev
```

### 打包
```
pnpm run build
```
